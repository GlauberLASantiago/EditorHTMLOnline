# 💻 GlauWeb Html Editor

O **GlauWeb Html Editor** é uma aplicação web interativa desenvolvida para facilitar a criação e edição de páginas HTML diretamente no navegador. O projeto reúne, em uma mesma interface, três áreas principais: edição do código-fonte, edição visual rica e pré-visualização em tempo real, oferecendo uma experiência completa para quem deseja escrever, testar e exportar conteúdos em HTML.

A proposta é unir praticidade e acessibilidade em um ambiente de edição que atende tanto usuários que preferem manipular o código diretamente quanto aqueles que desejam editar visualmente o conteúdo.

## ✨ Funcionalidades

- Editor de **código HTML** em texto puro
- Editor **visual rico** com formatação de conteúdo
- **Preview em tempo real** do resultado final
- Alternância de visualização entre:
  - desktop
  - celular
- Barra de ferramentas com recursos de formatação:
  - negrito
  - itálico
  - sublinhado
  - alinhamento
  - listas
  - títulos
  - parágrafo
  - links
  - remoção de formatação
- Seleção de fontes e tamanhos
- Aplicação de cores de texto com paletas prontas
- Inserção de símbolos musicais
- Salvamento do conteúdo como arquivo `.html`
- Interface dividida em três colunas para fluxo de edição completo

## 🎯 Objetivo do Projeto

O projeto foi criado para oferecer uma ferramenta de autoria HTML simples, visual e funcional, permitindo que usuários criem páginas web sem depender de softwares externos. A aplicação pode ser útil tanto em contextos educacionais quanto em produção rápida de páginas, materiais didáticos, protótipos e conteúdos interativos.

## 🧩 Estrutura da Interface

A aplicação está organizada em três áreas principais:

### 1. Código HTML
Permite editar diretamente o conteúdo em HTML bruto.

### 2. Editor Visual
Oferece uma área de edição rica com barra de ferramentas para formatação.

### 3. Preview
Mostra o resultado final renderizado em tempo real, simulando visualização em computador ou celular.

## ▶️ Como usar

1. Abra a aplicação no navegador.
2. Edite o conteúdo em uma das duas formas disponíveis:
   - pelo código HTML
   - pelo editor visual
3. Acompanhe o resultado no painel de preview.
4. Use a barra de ferramentas para aplicar formatação ao texto.
5. Defina o nome do arquivo.
6. Clique em **Salvar** para exportar o conteúdo como `.html`.

## 🎨 Recursos do Editor Visual

O editor inclui suporte a:

- títulos `h1`, `h2`...
- parágrafos
- listas ordenadas e não ordenadas
- links
- escolha de fonte
- escolha de tamanho
- cor de texto
- alinhamento
- símbolos musicais
- limpeza de formatação

Esses recursos tornam a ferramenta especialmente útil para produção de conteúdos educacionais, artísticos e musicais.

## 👀 Preview Responsivo

A área de pré-visualização permite alternar entre:

- **PC**
- **CEL**

Isso ajuda a testar a aparência do conteúdo em diferentes formatos de tela antes de salvar o arquivo final.

## 🎼 Diferenciais

Entre os diferenciais do projeto estão:

- integração entre edição visual e código-fonte;
- suporte a símbolos musicais;
- salvamento direto em HTML;
- preview instantâneo em iframe;
- interface leve e amigável.

A aplicação foi desenhada para funcionar em:
